es6初体验

发表于 2017-01-05

箭头函数

  1. 自身没有this，this指向上级
  2. 不能使用构造函数
  3. 不能使用argument ，用…代替
  4. 语法：
    • 形参 => 代码体

let

  • 不允许重复声明
  • 块级作用域
  • 没有预解析
  • 存在暂时性死区
  • 全局变量不再是window属性

const(同上)

  • 值不容许修改
  • 定义变量必须给值

``字符串

…运算符

对象

es5面向对象的复习

  1. 构造函数首字母大写
  2. es5及之前js是没有类的概念的
  3. 类是抽象的

静态和实例成员

  • 实例成员：供实例使用的属性的方法
  • 静态成员：供类使用的属性方法

class类语法

Object 对象（es6）

  • Object.assign(a,b);

    作用：把后面对象的第一层属性copy给第一个对象

    a继承b内容

  • Object.keys

    作用：获取对象所有的key

  • Object.values

    作用：获取对象所有的value

filter

es5方法filter,可以用来寻找多个符合条件的值

set

set 介绍：新增的构造函数，用于创建新数据结构的集合对象，特点是值唯一

  • size 去重（集合成员数量）
  • add 添加新值
  • delte删除值
  • has判断有没有某个值
  • clear清空所有值

for of

for of 语句是es6新增的循环语句

作用：用来遍历数组以及内置的伪数组结构中的值，另外Set Map 也支持

Map

传统的对象，不管你使用什么方式添加key，最终都会转为字符串

es6新增了一种数据结构,Map,类似与传统对象,特点是key可以是任意东西

  • set 添加值

  • get 获取值

Promise

使用Promise

  1. 第一步: new Promise 需要传一个回调函数, 这个函数供promise使用,promise在使用它的时候会传入两个方法yes与no
  2. 第二步: 调用实例的then方法做事情,这个then添加的方法会在承诺生效后执行

  • promise 实例创建好之后，执行我们传入的回调，执行这个回调是为了给回调提供yes与no方法，调用了yes执行then，调用no执行所有的then或catch.

async

强制按顺序执行
